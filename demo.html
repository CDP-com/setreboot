Run

This App allows you to change how automatic reboot behaves with Automatic Updates.

The Set Reboot App functionality is effective on Microsoft Windows XP Professional and newer Windows operating systems.

This App allows you to change how reboot behaves after an update. This is done by changing or adding a Windows registry setting for Automatic Updates:

It effects the default behavior of updates for the following situations:

Read More...
Logged in User Automatic Reboots Suppressed Automatic Reboots Allowed (default)
Nobody Automatic reboot Automatic reboot
User with Administrative Privileges Notification with no countdown occurs. User must initiate the reboot. Notification with a 5 minute countdown to automatic reboot occurs.
User without administrative privileges except for restart privilege Notification with no countdown occurs. User must initiate the reboot. Notification with a 5 minute countdown to automatic reboot occurs.
User without administrative privileges Notification with no countdown occurs. User must wait for an authorized user to reboot the computer. Notification with a 5 minute countdown to automatic reboot occurs.
Administrator with other users logged in Notification without countdown occurs. Administrator must initiate reboot. Notification with a 5 minute countdown to automatic reboot occurs.
Non-administrator with restart privilege while other users are logged on Notification without countdown occurs. User must initiate reboot. Notification with a 5 minute countdown to automatic reboot occurs.
Non-administrator without restart privilege while other users are logged on Notification with no countdown occurs. User must wait for an authorized user to reboot the computer. Notification with a 5 minute countdown to automatic reboot occurs.

Script Source Code

Q: What does this app do?
A: This app allows you to change/modify the behavior of your system's automatic updates.

Q: Who is this app for?
A: This app is for everyone with a Windows machine and who wants to change how updates are applied.
